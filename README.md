# Sales-Performance-Profitability-Analysis

## Project Overview
This project analyzes sales, profit, and customer purchasing patterns using Power BI and Python. The dashboard provides business insights through interactive visualizations and DAX measures.

## Project Architecture

Dataset (CSV)
        ↓
Data Cleaning (Python)
        ↓
Data Transformation (Power Query)
        ↓
Data Modeling (Power BI)
        ↓
DAX Measures
        ↓
Interactive Dashboard
        ↓
Business Insights

## Tools Used
- Power BI
- Python
- Pandas
- NumPy
- Matplotlib
- SQL
- DAX

## Project Files
- Sales Performance Profitability Analysis Report (PDF)
- Power BI Dashboard (.pbix)
- Dataset (.csv)
- Python Code (.ipynb/.py)

## Dashboard Features
- Total Sales
- Total Profit
- Total Quantity
- Profit Percentage
- YTD Sales
- Sales by Category
- Sales by Region
- Sales by Ship Mode

## Conclusion
This project helps analyze business performance and supports data-driven decision making using Power BI dashboards and Python-based data analysis.
